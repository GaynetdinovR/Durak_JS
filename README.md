# Durak_JS
Дурак - карточная игра для 2-8 игроков

## Стек технологий

+ HTML5
+ SASS
+ WEBPACK
+ JAVASCRIPT

## Установка

+ Клонировать проект

```
git clone https://github.com/GaynetdinovR/Durak_JS.git
```

+ Установить необходимые пакеты

```
npm install
```

+ Запустить локальный сервер

```
npm run serve
```
